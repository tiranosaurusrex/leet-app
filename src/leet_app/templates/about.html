What is LEET?

LEET (London Energy Efficiency Tool) is an innovative web platform designed to empower users with actionable insights into London's energy consumption patterns. By providing borough-level energy demand data, LEET enables users to assess building energy efficiency, compare consumption across sectors, and support sustainability efforts across the city.

How it Works

LEET consolidates and visualizes energy data from reliable public sources and integrates The Guardian Open Platform API to keep users informed on sustainability topics. Through interactive visualizations like bar charts, pie charts, and geo-maps, users can explore energy demand patterns, enabling more informed decisions regarding energy usage and building retrofitting strategies.

Technologies Used
  • Flask - A micro web framework for Python.
  • Plotly - For creating interactive charts and visualizations.
  • SQLAlchemy - A powerful database toolkit for Python.
  • API Integration - Powered by The Guardian Open Platform API for sustainability-related content.
